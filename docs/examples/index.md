## Overview

This section contains practical examples demonstrating how to use xilikelihood
for likelihood evaluation and simulation validation.

## Available Examples

- **Likelihood Analysis**: Main likelihood setup, deterministic mock data, and
  copula/Gaussian likelihood evaluation.
- **Basic Simulation**: Correlation-function simulations for validation and
  comparison workflows. This path requires the simulation dependencies.
